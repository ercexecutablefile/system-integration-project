# System Integration Project

## Overview
This project demonstrates system integration using:
- One Web-Based System
- One Desktop System
- One centralized API system
- One shared MySQL database

## Architecture

Web System  →  
             →  API System → MySQL Database  
Desktop System →

## Tech Stack
- API: PHP (or Laravel)
- Web: PHP / HTML / JS
- Desktop: C# WinForms (or Python)
- Database: MySQL
- Testing: Postman

## Folder Structure
- api/ → Backend API (CRUD operations)
- web-system/ → Web client consuming API
- desktop-system/ → Desktop client consuming API
- database/ → SQL schema and dumps
- docs/ → API documentation and diagrams

## Features
- Create, Read, Update, Delete (CRUD)
- Centralized database
- API-based communication only
- Postman testing supported
